    ```markdown
    ## 资本主义经济
    
    ### 经济是什么
    
    经济是指，生活中必要的物品，即**商品（生产物）**，以及满足人们需求的服务（医疗、教育、保险、运输等）的生产、交换、流通、分配、消费活动。
    
    
    ### 资本主义经济的成立
    
    18世纪后半叶，英国的工业革命（p.80）为契机，资本主义经济得以确立。通过英法等国的近代市民革命（p.76,78），在19世纪初，资本主义经济在欧洲和美国迅速发展。
    
    
    ### 资本主义经济的特征
    
    资本主义经济是指，资本家拥有生产资料（工厂、土地、机器、资本），劳动者提供劳动力，以追求利润（利益）为目标的经济体制。
    
    
    ### 资本主义经济的特点
    
    1. **生产资料的私有制**: 生产资料为个人所有。
    2. **自由竞争**: 经济活动自由进行，不受政府干预。
    3. **市场经济**: 生产的商品和服务都在市场上买卖。
    4. **利润追求**: 经济活动以追求利润为目的。
    
    
    ### 资本主义经济的问题点
    
    在资本主义经济中，资本家和劳动者之间存在对立，自由竞争导致企业倒闭，从而引发失业问题。此外，经济还呈现出繁荣—衰退—萧条—复苏的周期性波动（p.138）。由于自由竞争，成功的企业会垄断市场和财富，从而导致资本积累和集中，最终形成垄断市场（p.134）。
    
    ```

    ```markdown
    ## 社会主义经济
    
    ### 社会主义经济的成立
    
    在资本主义经济发展中，资本家与劳动者之间的矛盾日益加剧。19世纪中期，马克思的《资本论》的出现，为社会主义提供了理论基础。同时，社会主义运动也兴起，最终在俄国十月革命后，第一个社会主义国家诞生（p.101）。
    
    
    ### 社会主义经济的特征
    
    社会主义经济指的是，资本家和劳动者之间的对立较小的平等社会实现的经济体制。
    
    
    ### 社会主义经济的模式
    
    1. **生产资料的公有制**: 生产资料为国家所有，劳动者共同享有。
    2. **计划经济**: 国家掌控经济活动，以计划经济的方式进行，自由竞争受到限制。例如，第一个五年计划（1928-32年）和第二个五年计划（1933-37年）的实施。
    3. **利润的限制**: 利润归国家或集体所有，劳动者平均分配。
    
    
    ### 社会主义经济的问题点与对策
    
    社会主义计划经济的不完善和生产资料的缺乏，导致物资短缺和生产效率低下，经济发展缓慢。为了解决这个问题，一些国家开始尝试市场经济的机制，并逐步引入市场调节机制。例如，苏联、中国等国家都尝试了市场化改革。
    
    **国家案例：**
    
    * **旧苏联**:  戈尔巴乔夫的改革（1985-91年）……市场经济的引入
    * **中国**: 改革开放政策（1978年）……生产责任制、经济特区的设置→外国企业的涌入
    * **越南**:  Doi Moi（革新）政策（1986年）……市场经济的引入，外国资本的涌入
    ```

  ```markdown
  ## 需要与供給的法则
  
  **需要(Demand)的法则**
  
  某种商品的价格上升，其需求量就会减少；反之，商品价格下降，其需求量就会增加。这被称为**需要法则**。用图表表示，它是一条右下倾斜的曲线，被称为**需求曲线**(DD曲线)。
  
  **供給(Supply)的法则**
  
  某种商品的价格上升，其供給量就会增加；反之，商品价格下降，其供給量就会减少。这被称为**供給法则**。用图表表示，它是一条右上倾斜的曲线，被称为**供給曲线**(SS曲线)。
  
  
  **如何解读需要和供給曲线**
  
  * **需求曲线**:  图中需求曲线(D)表示在不同的价格水平下，消费者愿意购买的商品数量。曲线向右下方倾斜，说明价格越高，需求量越少；价格越低，需求量越多。
  
  * **供給曲线**: 图中供給曲线(S)表示在不同的价格水平下，生产者愿意提供的商品数量。曲线向右上方倾斜，说明价格越高，供給量越多；价格越低，供給量越少。
  
  
  ```

    ```markdown
    ## 价格机制
    
    ### 价格高的时候 → 供給 > 需要
    
    价格高(P₂)时，需求量减少(Q₁)。由于需求量减少，**市场过剩**产生(Q₄-Q₁)。为了卖完多余商品，供給量减少。结果，需求量增加，供給量减少。
    
    
    ### 价格低的时候 → 需要 > 供給
    
    价格低(P₁)时，需求量增加(Q₃)。由于需求量增加，**商品不足**(超额需求)产生(Q₃-Q₂)。为了获得更多利润，供給量增加。结果，需求量减少，供給量增加。
    
    
    ### 均衡价格
    
    由于以上原因，无论价格高还是低，价格都会向P₀靠拢。当价格为P₀时，DD曲线和SS曲线均衡点(E点)重合，需要和供給一致。像这样，价格的上下波动会使需要和供給互相靠近的过程，被称为**自发调整機能**，亚当·史密斯称之为“看不见的手”。
    
    **价格机制运作解释:**
    
    价格机制是市场经济中商品价格的形成和变动机制。通过供求关系的相互作用来调节商品的生产、分配和消费。
    
    * **需求**: 消费者对某种商品的需求量与其价格呈反比关系，价格越高，需求量越低；价格越低，需求量越高。
    
    * **供給**: 生产者对某种商品的供給量与其价格呈正比关系，价格越高，供給量越高；价格越低，供給量越低。
    
    * **均衡价格 (P₀):** 当需求量等于供給量时的价格。在均衡价格下，市场实现供求平衡，没有过剩或短缺。
    
    * **价格波动:** 当市场价格高于均衡价格时，会产生过剩，导致价格下降；当市场价格低于均衡价格时，会产生短缺，导致价格上涨。价格的波动正是市场机制自我调整的过程，最终会趋向均衡价格。
    
    * **自发调整功能:** 市场价格的波动并不是混乱的，而是遵循着一定的规律。通过价格信号，市场自动调节供求关系，最终达到均衡状态。这就是市场机制的自发调整功能，它无需政府干预就能有效配置资源。
    
    
    ```

    ```markdown
    ## 需要曲线・供給曲線のシフト (曲线移动)
    
    **需要曲線のシフト (DD→D'D')**
    
    需要曲線右方シフト(DD→D'D')，需求量增加(Q₀→Q₂)，价格上涨(P₀→P₂)。这是以下几种情况：
    
    1. 消费者的所得上升或易于获得贷款时自由地使用
    2. 商品价格下跌时，替代商品需求增加
    3. 互补商品(替代品：大米、咖啡和茶)价格上涨时
    4.  促销活动使商品(例如：汉堡包、披萨、咖啡+糕点)的需求量下降时
    
    
    需要曲線左方シフト (DD→D''D'')，需求量减少(Q₀→Q₁)，价格下跌(P₀→P₁)。这是1-4的逆向情况。
    
    
    
    **供給曲線のシフト (SS→S'S')**
    
    供給曲線右方シフト (SS→S'S')，供給量增加(P₀→P₁)。这是以下几种情况：
    
    1. 原材料价格下降时
    2. 新的生产技术的开发使得生产成本下降，从而导致生产增加
    3. 提高生产效率时
    4. 劳动成本下降时
    5. 技术进步使生产成本下降时
    
    
    供給曲線左方シフト (SS→S''S'')，供給量减少(Q₀→Q₁)，价格上涨(P₀→P₂)。这是1-5的逆向情况。
    
    
    **需要和供給曲线位移的解释:**
    
    曲线位移表示影响需要或供給的因素发生了变化，导致在任何给定价格下，需求量或供給量发生变化。  这与沿着曲线的移动不同，后者是由于价格自身的变化引起的需要或供給量变化。
    
    * **需要曲线位移的原因:**  消费者收入、消费者偏好、替代品价格、互补品价格、消费者预期等等因素的变化都会引起需要曲线的位移。
    
    * **供給曲线位移的原因:** 生产技术、投入价格（如原材料、劳动力）、政府政策（如税收、补贴）、生产者预期等等因素的变化都会引起供給曲线的位移。
    
    
    ```

    ```markdown
    ## 需要的价格弹力性
    
    **价格变化对需求量影响较小的需要曲线 (D₁) **
    
    对于日常必需品或替代品较多的商品来说，价格变化(P₂-P₁)对需求量变化(Q₃-Q₂)的影响较小。因此，**需要价格弹性较小**。
    
    
    **价格变化对需求量影响较大的需要曲线 (D₂) **
    
    对于奢侈品或替代品较少的商品来说，价格变化(P₂-P₁)对需求量变化(Q₄-Q₁)的影响较大。因此，**需要价格弹性较大**。

    **价格变化对供給量影响较小的供給曲线 (S₁) **
    
    对于难以迅速调整产量的产品来说，价格变化(P₂-P₁)对供給量变化(Q₃-Q₂)的影响较小。因此，**供給价格弹性较小**。
    
    
    **价格变化对供給量影响较大的供給曲线 (S₂) **
    
    对于能够迅速调整产量的工业产品来说，价格变化(P₂-P₁)对供給量变化(Q₄-Q₁)的影响较大。因此，**供給价格弹性较大**。
    
    
    **需要和供給价格弹性解释:**
    
    价格弹性衡量的是价格变化对需求量或供給量的影响程度。弹性值越大，表示价格变化对需求量或供給量的影响越大；弹性值越小，表示价格变化对需求量或供給量的影响越小。
    
    * **需要价格弹性:**  反映的是消费者对价格变化的敏感程度。奢侈品的需求价格弹性通常较大，因为消费者对价格变化较为敏感。而必需品的需求价格弹性通常较小，因为消费者对价格变化相对不敏感。
    
    * **供給价格弹性:** 反映的是生产者对价格变化的反应程度。那些能够迅速调整产量的行业（如制造业）的供給价格弹性通常较大，而那些难以迅速调整产量的行业（如农业）的供給价格弹性通常较小。
    ```

- **市场的失败**

  - 独占+公共财+外部性+情报非对称

    ![image-20241102150020405](https://cdn.jsdelivr.net/gh/bronyowl/typopicbed@main/img/image-20241102150020405.png)

    ```markdown
    ## 市场的失败
    
    市场的失败是指市场机制由于某种原因导致需要和供給无法有效协调的情况。常见的市场失败包括以下几种：
    
    **公共財的問題**
    
    道路、桥梁、公园等社会资本和警察、消防等公共服务被定义为公共財。由于公共財无法排除他人享用且收益无法被有效地分配，市场无法有效地提供公共財，因此需要政府提供资金进行干预。
    
    
    **外部不経済と外部経済**
    
    **外部不经济**是指由于某些活动（如大气污染、噪音等）造成的环境破坏等负面影响，这些负面影响并未被完全纳入市场价格之中，导致市场过度生产或消费。这会导致社会福利下降，需要政府干预进行调节和管控。
    
    另一方面，**外部经济**是指某些活动带来的积极外部效应（如新式建筑带来的地区经济活化等），但这些积极的外部效应并没有完全反映在市场价格中，导致市场低估了这些活动的价值。同样需要政府干预来促进这些积极的外部效应。
    
    
    ```
  
  - 其中外部性需要好好了解
