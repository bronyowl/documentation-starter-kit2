    ```markdown
    ## 资本主义经济
    
    ### 经济是什么
    
    经济是指，生活中必要的物品，即**商品（生产物）**，以及满足人们需求的服务（医疗、教育、保险、运输等）的生产、交换、流通、分配、消费活动。
    
    
    ### 资本主义经济的成立
    
    18世纪后半叶，英国的工业革命（p.80）为契机，资本主义经济得以确立。通过英法等国的近代市民革命（p.76,78），在19世纪初，资本主义经济在欧洲和美国迅速发展。
    
    
    ### 资本主义经济的特征
    
    资本主义经济是指，资本家拥有生产资料（工厂、土地、机器、资本），劳动者提供劳动力，以追求利润（利益）为目标的经济体制。
    
    
    ### 资本主义经济的特点
    
    1. **生产资料的私有制**: 生产资料为个人所有。
    2. **自由竞争**: 经济活动自由进行，不受政府干预。
    3. **市场经济**: 生产的商品和服务都在市场上买卖。
    4. **利润追求**: 经济活动以追求利润为目的。
    
    
    ### 资本主义经济的问题点
    
    在资本主义经济中，资本家和劳动者之间存在对立，自由竞争导致企业倒闭，从而引发失业问题。此外，经济还呈现出繁荣—衰退—萧条—复苏的周期性波动（p.138）。由于自由竞争，成功的企业会垄断市场和财富，从而导致资本积累和集中，最终形成垄断市场（p.134）。
    
    ```

    ```markdown
    ## 社会主义经济
    
    ### 社会主义经济的成立
    
    在资本主义经济发展中，资本家与劳动者之间的矛盾日益加剧。19世纪中期，马克思的《资本论》的出现，为社会主义提供了理论基础。同时，社会主义运动也兴起，最终在俄国十月革命后，第一个社会主义国家诞生（p.101）。
    
    
    ### 社会主义经济的特征
    
    社会主义经济指的是，资本家和劳动者之间的对立较小的平等社会实现的经济体制。
    
    
    ### 社会主义经济的模式
    
    1. **生产资料的公有制**: 生产资料为国家所有，劳动者共同享有。
    2. **计划经济**: 国家掌控经济活动，以计划经济的方式进行，自由竞争受到限制。例如，第一个五年计划（1928-32年）和第二个五年计划（1933-37年）的实施。
    3. **利润的限制**: 利润归国家或集体所有，劳动者平均分配。
    
    
    ### 社会主义经济的问题点与对策
    
    社会主义计划经济的不完善和生产资料的缺乏，导致物资短缺和生产效率低下，经济发展缓慢。为了解决这个问题，一些国家开始尝试市场经济的机制，并逐步引入市场调节机制。例如，苏联、中国等国家都尝试了市场化改革。
    
    **国家案例：**
    
    * **旧苏联**:  戈尔巴乔夫的改革（1985-91年）……市场经济的引入
    * **中国**: 改革开放政策（1978年）……生产责任制、经济特区的设置→外国企业的涌入
    * **越南**:  Doi Moi（革新）政策（1986年）……市场经济的引入，外国资本的涌入
    ```

  ```markdown
  ## 需要与供給的法则
  
  **需要(Demand)的法则**
  
  某种商品的价格上升，其需求量就会减少；反之，商品价格下降，其需求量就会增加。这被称为**需要法则**。用图表表示，它是一条右下倾斜的曲线，被称为**需求曲线**(DD曲线)。
  
  **供給(Supply)的法则**
  
  某种商品的价格上升，其供給量就会增加；反之，商品价格下降，其供給量就会减少。这被称为**供給法则**。用图表表示，它是一条右上倾斜的曲线，被称为**供給曲线**(SS曲线)。
  
  
  **如何解读需要和供給曲线**
  
  * **需求曲线**:  图中需求曲线(D)表示在不同的价格水平下，消费者愿意购买的商品数量。曲线向右下方倾斜，说明价格越高，需求量越少；价格越低，需求量越多。
  
  * **供給曲线**: 图中供給曲线(S)表示在不同的价格水平下，生产者愿意提供的商品数量。曲线向右上方倾斜，说明价格越高，供給量越多；价格越低，供給量越少。
  
  
  ```

    ```markdown
    ## 价格机制
    
    ### 价格高的时候 → 供給 > 需要
    
    价格高(P₂)时，需求量减少(Q₁)。由于需求量减少，**市场过剩**产生(Q₄-Q₁)。为了卖完多余商品，供給量减少。结果，需求量增加，供給量减少。
    
    
    ### 价格低的时候 → 需要 > 供給
    
    价格低(P₁)时，需求量增加(Q₃)。由于需求量增加，**商品不足**(超额需求)产生(Q₃-Q₂)。为了获得更多利润，供給量增加。结果，需求量减少，供給量增加。
    
    
    ### 均衡价格
    
    由于以上原因，无论价格高还是低，价格都会向P₀靠拢。当价格为P₀时，DD曲线和SS曲线均衡点(E点)重合，需要和供給一致。像这样，价格的上下波动会使需要和供給互相靠近的过程，被称为**自发调整機能**，亚当·史密斯称之为“看不见的手”。
    
    **价格机制运作解释:**
    
    价格机制是市场经济中商品价格的形成和变动机制。通过供求关系的相互作用来调节商品的生产、分配和消费。
    
    * **需求**: 消费者对某种商品的需求量与其价格呈反比关系，价格越高，需求量越低；价格越低，需求量越高。
    
    * **供給**: 生产者对某种商品的供給量与其价格呈正比关系，价格越高，供給量越高；价格越低，供給量越低。
    
    * **均衡价格 (P₀):** 当需求量等于供給量时的价格。在均衡价格下，市场实现供求平衡，没有过剩或短缺。
    
    * **价格波动:** 当市场价格高于均衡价格时，会产生过剩，导致价格下降；当市场价格低于均衡价格时，会产生短缺，导致价格上涨。价格的波动正是市场机制自我调整的过程，最终会趋向均衡价格。
    
    * **自发调整功能:** 市场价格的波动并不是混乱的，而是遵循着一定的规律。通过价格信号，市场自动调节供求关系，最终达到均衡状态。这就是市场机制的自发调整功能，它无需政府干预就能有效配置资源。
    
    
    ```

    ```markdown
    ## 需要曲线・供給曲線のシフト (曲线移动)
    
    **需要曲線のシフト (DD→D'D')**
    
    需要曲線右方シフト(DD→D'D')，需求量增加(Q₀→Q₂)，价格上涨(P₀→P₂)。这是以下几种情况：
    
    1. 消费者的所得上升或易于获得贷款时自由地使用
    2. 商品价格下跌时，替代商品需求增加
    3. 互补商品(替代品：大米、咖啡和茶)价格上涨时
    4.  促销活动使商品(例如：汉堡包、披萨、咖啡+糕点)的需求量下降时
    
    
    需要曲線左方シフト (DD→D''D'')，需求量减少(Q₀→Q₁)，价格下跌(P₀→P₁)。这是1-4的逆向情况。
    
    
    
    **供給曲線のシフト (SS→S'S')**
    
    供給曲線右方シフト (SS→S'S')，供給量增加(P₀→P₁)。这是以下几种情况：
    
    1. 原材料价格下降时
    2. 新的生产技术的开发使得生产成本下降，从而导致生产增加
    3. 提高生产效率时
    4. 劳动成本下降时
    5. 技术进步使生产成本下降时
    
    
    供給曲線左方シフト (SS→S''S'')，供給量减少(Q₀→Q₁)，价格上涨(P₀→P₂)。这是1-5的逆向情况。
    
    
    **需要和供給曲线位移的解释:**
    
    曲线位移表示影响需要或供給的因素发生了变化，导致在任何给定价格下，需求量或供給量发生变化。  这与沿着曲线的移动不同，后者是由于价格自身的变化引起的需要或供給量变化。
    
    * **需要曲线位移的原因:**  消费者收入、消费者偏好、替代品价格、互补品价格、消费者预期等等因素的变化都会引起需要曲线的位移。
    
    * **供給曲线位移的原因:** 生产技术、投入价格（如原材料、劳动力）、政府政策（如税收、补贴）、生产者预期等等因素的变化都会引起供給曲线的位移。
    
    
    ```

    ```markdown
    ## 需要的价格弹力性
    
    **价格变化对需求量影响较小的需要曲线 (D₁) **
    
    对于日常必需品或替代品较多的商品来说，价格变化(P₂-P₁)对需求量变化(Q₃-Q₂)的影响较小。因此，**需要价格弹性较小**。
    
    
    **价格变化对需求量影响较大的需要曲线 (D₂) **
    
    对于奢侈品或替代品较少的商品来说，价格变化(P₂-P₁)对需求量变化(Q₄-Q₁)的影响较大。因此，**需要价格弹性较大**。

    **价格变化对供給量影响较小的供給曲线 (S₁) **
    
    对于难以迅速调整产量的产品来说，价格变化(P₂-P₁)对供給量变化(Q₃-Q₂)的影响较小。因此，**供給价格弹性较小**。
    
    
    **价格变化对供給量影响较大的供給曲线 (S₂) **
    
    对于能够迅速调整产量的工业产品来说，价格变化(P₂-P₁)对供給量变化(Q₄-Q₁)的影响较大。因此，**供給价格弹性较大**。
    
    
    **需要和供給价格弹性解释:**
    
    价格弹性衡量的是价格变化对需求量或供給量的影响程度。弹性值越大，表示价格变化对需求量或供給量的影响越大；弹性值越小，表示价格变化对需求量或供給量的影响越小。
    
    * **需要价格弹性:**  反映的是消费者对价格变化的敏感程度。奢侈品的需求价格弹性通常较大，因为消费者对价格变化较为敏感。而必需品的需求价格弹性通常较小，因为消费者对价格变化相对不敏感。
    
    * **供給价格弹性:** 反映的是生产者对价格变化的反应程度。那些能够迅速调整产量的行业（如制造业）的供給价格弹性通常较大，而那些难以迅速调整产量的行业（如农业）的供給价格弹性通常较小。
    ```

- **市场的失败**

  - 独占+公共财+外部性+情报非对称

    ![image-20241102150020405](https://cdn.jsdelivr.net/gh/bronyowl/typopicbed@main/img/image-20241102150020405.png)

    ```markdown
    ## 市场的失败
    
    市场的失败是指市场机制由于某种原因导致需要和供給无法有效协调的情况。常见的市场失败包括以下几种：
    
    **公共財的問題**
    
    道路、桥梁、公园等社会资本和警察、消防等公共服务被定义为公共財。由于公共財无法排除他人享用且收益无法被有效地分配，市场无法有效地提供公共財，因此需要政府提供资金进行干预。
    
    
    **外部不経済と外部経済**
    
    **外部不经济**是指由于某些活动（如大气污染、噪音等）造成的环境破坏等负面影响，这些负面影响并未被完全纳入市场价格之中，导致市场过度生产或消费。这会导致社会福利下降，需要政府干预进行调节和管控。
    ```

    ![image-20241102150145553](https://cdn.jsdelivr.net/gh/bronyowl/typopicbed@main/img/image-20241102150145553.png)

    ```markdown
    ## GDP和GNP
    
    国的1年的经济活动的量（大小）衡量的指标， 国内生产总值（GDP）和国民生产总值（GNP）。
    
    ### 国内生产总值（GDP）=国内的生产要素的总额
    
    生产总值的国内部分，在一年内国内生产的付加价值的总和，是从生产要素的中间生产物的总额中扣除的。它是衡量国家经济活动规模的指标，在世界范围内被广泛使用。
    
    ### 国民生产总值（GNP）=GDP+海外的净所得
    
    国民一年内在国内外生产的总产值，是GDP加上海外的净所得（从国外获得的所得-向国外支付的款项）的总和。国民所得（GNI）与国民生产总值（GNP）数值相同。
    
    从GNP中扣除折旧费和损耗等，剩下的称为国民净生产值（NNP）。此外，从NNP中扣除包含在税收和政府为保障・育成产业而支出的补助金，剩下的，称为国民所得(NI)。
    ```

    ```markdown
    ##  GDP、GNP、NNP和NI的构成图解
    
    这张图以图解的形式展示了国内生产总值（GDP）、国民生产总值（GNP）、国民净生产总值（NNP）和国民所得（NI）之间的关系及其构成要素。通过图表，我们可以清晰地看到各个指标之间的递减关系，以及构成这些指标的要素。

    * **国内的纯生产额:**  这是指在一个国家境内生产的最终产品的总价值，不包括中间产品的价值。
    
    * **最终生产额:** 指最终产品和服务的总价值。
    
    * **海外的净所得:** 指一个国家从海外获得的收入（例如，本国公司在海外的投资收益）减去支付给海外的款项（例如，支付给外国工人的工资）的差额。
    
    * **中间生产物的总额:** 指生产最终产品所使用的中间产品的总价值。
    
    * **固定资本减耗:** 指由于生产活动导致的资本损耗。
    
    * **间接税-补贴:**  指政府征收的间接税（例如，销售税）减去政府提供的补贴。
    
    
    **指标之间的关系:**
    
    * **GDP = 国内生产总值:** 只计算在国境内生产的最终产品和服务的价值。
    
    * **GNP = 国民生产总值:**  在GDP的基础上，加上来自国外的净所得（海外的净所得）。
    
    * **NNP = 国民净生产总值:**  从GNP中扣除固定资本减耗，即扣除了生产过程中机器设备等折旧的部分，更准确地反映了一国实际生产能力。
    
    * **NI = 国民所得:** 从NNP中扣除间接税，加上补贴，反映了居民和企业从生产活动中实际获得的收入。
    
    
    **总结:**
    
    这张图形象地说明了四个关键经济指标之间的关系，从国内生产总值到国民所得，逐步扣除不同的项目，最终得到反映居民和企业实际收入的国民所得。通过这种递减的逻辑，可以更深入地理解这些指标的含义以及它们之间的联系。
    ```

    ```markdown
    **三面等价原则的解释:**
    
    三面等价原则指出，一个经济体的总产出（生产）、总收入（分配）和总支出（支出）在理论上应该相等。 这是因为：
    
    1. **生产的产品最终会被卖出去（支出）：** 生产出来的商品和服务最终会被消费者或企业购买，形成总支出。
    
    2. **销售收入最终会作为报酬或利润分配出去（分配）：**  企业销售商品和服务的收入，一部分会作为工资、租金、利息等支付给生产要素的拥有者（雇佣者报酬、财产所得），另一部分作为企业利润。
    
    3. **支出最终会转化为对商品和服务的总需求（生产）：** 消费者的消费支出和企业的投资支出最终会转化为对商品和服务的总需求，驱动经济的生产。
    
    因此，从生产、分配和支出的三个角度看一个经济体的活动，虽然视角不同，但结果应该是一致的，即总价值相等。  这有助于对一个经济体的整体运行情况有更全面的认识，并对各个经济指标进行更准确的分析。  需要注意的是，这只是一个理论模型，实际经济中可能存在一些统计上的差异。
    ```

    ```markdown
    ## 景气变动
    
    **资本主义经济**存在着景气变动（景气循环），经历着**好转→衰退→不景气→回暖**的循环。
    
    **图表说明：**
    
    * **纵轴：** 生产量
    * **横轴：** 时间
    * **①好转：**  需求增加，投资和生产扩大，从而带动产出增加。由于供不应求，价格上涨，形成扩张。
    * **②衰退：** 生产逐渐减少，价格下降，生产减少。 倒闭和失业增加。
    * **③不景气：** 生产停滞不前，倒闭和失业进一步增加。企业注重调整库存（低库存的阶段）。价格下跌，导致通货紧缩。
    * **④回暖：**  库存调整结束，生产开始回升。失业减少，需求回升。
    ```

    ```markdown
    ## 经济成长率的计算方法
    
    ### 名义经济成长率
    
    名义经济成长率的计算公式如下：
    
    名义经济成长率 = (今年GDP - 前年GDP) ÷ 前年GDP × 100 (%)
    
    
    **计算步骤示例：**
    
    假设前一年的名义GDP为100亿元，今年的名义GDP为150亿元。则名义经济成长率为：(150 - 100) ÷ 100 × 100 = 50%。
    
    
    ### 实际经济成长率
    
    为了计算实际经济成长率，首先需要计算实际GDP。计算公式如下：
    
    实际GDP = 名义GDP ÷ GDP平减指数 × 100
    
    **其中：**
    
    * GDP平减指数：衡量名义GDP中物价变动部分的指数。(名义GDP扣除物价变动因素后得到的实际GDP)
    
    
    **计算实际经济成长率的公式：**
    
    实际经济成长率 = (今年的实际GDP - 前年的实际GDP) ÷ 前年的实际GDP × 100 (%)
    
    **计算步骤示例：**
    
    假设前一年的GDP平减指数为100，今年的GDP平减指数为105。
    那么，前一年的实际GDP为：(100 ÷ 100) × 100 = 100 亿元。
    今年的实际GDP为：(150 ÷ 105) × 100 = 143 亿元。
    因此，实际经济成长率为：(143 - 100) ÷ 100 × 100 = 43%。
    
    
    ### 名义GDP和实际GDP
    
    * **名义GDP:** 指的是用当年价格计算的全部最终产品和服务的市场价值。
    
    * **实际GDP:** 指的是用基年价格计算的全部最终产品和服务的市场价值，它消除了物价变动的影响，更能反映经济的实际增长。
    ```

  ```markdown
  ## 物价与物价指数
  
  **物价**是指各种商品（财货、服务）价格的总和，平均化后，以某一年为基准年，其物价为100，其他年份的物价以此为基准表示的数值称为物价指数。物价指数有企业物价指数和消费者物价指数两种，分别反映企业购买商品和消费者购买商品和服务的價格变动情况。
  
  
  ## 通货膨胀与通货紧缩
  
  当整体物价下跌，而商品价格上涨时，称为**通货膨胀**（或膨胀）。当整体物价上涨，而商品价格下跌时，称为**通货紧缩**（或紧缩）。
  
  此外，通货紧缩和通货膨胀会同时发生，被称为**滞胀**。在滞胀时期，商品价格通常会下降，但在1973年的第一次石油危机的影响下，石油价格急剧上涨，物价也随之上涨，由此在发达国家出现了滞胀。
  ```

    ```markdown
    ## 通货膨胀的原因
    
    ### 需求拉动型通货膨胀 (需求·拉引·インフレ)
    ```

    ```markdown
    ## 货币
    
    ### 货币的机能和役割
    
    货币是商品（财货）和服务交换（流通）的媒介。货币的机能包括：现金货币和信用货币两种。
    
    **现金货币:**
    
    * 支付手段
    * 价值尺度
    * 储藏手段
    
    **信用货币:**
    
    * 支付手段：方便快捷地进行支付的资金。
    * 储藏手段：为了方便将来使用而储蓄的资金。
    
    
    ### 货币的役割
    
    货币有四大功能：
    
    1. **价值尺度:**  衡量商品价值的标准。货币的价值是稳定的。
    2. **交换媒介:** 商品交换的方式。
    3. **价值储藏手段:**  可以保存和未来交易的价值。
    4. **支付手段:**  结算交易和债务支付的方式。
    
    
    ### 金本位制度
    
    金本位制度是指以金作为货币的基础制度，国家持有黄金储备并发行与黄金储备量挂钩的货币的制度。世界上最早采用金本位制度的是英国（1816年），日本也于1897年采用。
    
    金本位制度的优缺点：
    
    **优点：**
    
    * 货币稳定：由于以黄金为基础，货币价值稳定，通货膨胀不易发生。
    * 国际收支自动调节：当国际收支出现逆差时，黄金外流，自动调节国际收支。
    
    **缺点：**
    
    * 货币供应量有限：货币供应量受到黄金储备量的限制，难以适应经济变化的需求。
    
    
    **(图示说明)**
    
    **金本位制度下国际收支的自动调节作用:**
    
    **情况1：通货膨胀**
    
    1. **物价上涨**
    2. **资金流入**  (黄金流入，货币供给增加)
    3. **商品出口减少** (因为物价上涨，商品竞争力下降)
    4. **国际收支恶化**
    5. **通货紧缩**
    ```

      ```markdown
      ## 银行的役割
      
      银行主要有三大业务：
      
      1. **资金募集:** 从个人和企业筹集资金，收取利息。
      2. **资金融通:** 向企业和个人放款，收取利息。
      3. **信托业务:**  接受存款和公共资金，进行资金周转。
      
      
      ### 信用创造
      
      银行从众多存款人处吸收资金，并随时准备支付现金，因此银行始终持有部分现金。为了满足企业贷款的需求，银行会将这部分存款作为准备金，其余部分作为贷款发放给企业。准备金是银行为支付存款人随时提取存款而必须持有的现金或其等价物。
      
      **信用创造的计算方法:**
      
      **信用创造总额 = 本源的存款 × 支撑准备率分之一**
      
      **信用创造总额 = 存款余额 - 本源的存款**
      
      
      **其中：**
      
      * **本源的存款:** 指银行从存款人处直接获得的原始存款。
      
      * **支撑准备率:** 指的是银行为满足存款人随时提取存款需求而必须持有的准备金占存款总额的比例。
      
      
      **计算示例：**
      
      假设本源存款为100万元，法定准备率为10%。
      
      信用创造总额 = 100万元 × (1 ÷ 0.1) = 1000万元 (存款余额)
      
      信用创造总额 = 1000万元 - 100万元 = 900万元 (信用创造额)
      
      
      ```

      ```markdown
      ## 日本银行的金融政策
      
      日本银行通过调节国内流通货币量（货币供应）来实施稳定物价和调节经济的金融政策。具体而言，在经济景气时期减少货币量，在经济不景气时期增加货币量（金融宽松）政策。
      
      为实施金融政策，主要采用以下两种手段：
      
      ### 公开市场操作
      日本银行在金融市场上与商业银行进行国债、票据等有价证券（等同于具有价值的债券）的买卖，藉此调节国内流通货币量。
      - 在经济景气时期：向商业银行出售有价证券（卖出操作），吸收商业银行的货币以减少货币量
      - 在经济不景气时期：从商业银行购买有价证券（买入操作），增加货币量
      
      ### 存款准备金率操作
      商业银行依法需要将一定比例的存款无息存入日本银行，这一比例称为存款准备金率。
      
      日本银行通过调整存款准备金率来调节国内流通货币量：
      - 在经济景气时期：提高存款准备金率以减少放贷，从而减少货币量
      - 在经济不景气时期：降低存款准备金率以增加放贷，从而增加货币量
      
      ```
      
      政策效果对比
      
      | 操作类型         | 经济景气时期（通胀） | 经济不景气时期（通缩） |
      | ---------------- | -------------------- | ---------------------- |
      | 公开市场操作     | 卖出操作             | 买入操作               |
      | 存款准备金率操作 | 提高准备金率         | 降低准备金率           |
      | 结果             | 货币量减少           | 货币量增加             |

      ```markdown
      ## 税收的构成
      
      税收分为向国家缴纳的国税和向地方公共团体缴纳的地方税，这些又可分为直接税和间接税。
      
      直接税是由个人缴纳的所得税和企业缴纳的法人税等，由负有纳税义务的人直接负担的税收。间接税是消费税等包含在商品和服务价格中的税收，纳税人和负担者不同的税收。
      
      ### 税收种类表
      
      直接税：
      - 国税：所得税、法人税、相续税
      - 地方税：居民税、固定资产税
      
      间接税：
      - 国税：消费税、酒税、烟草税
      - 地方税：地方消费税
      
      ### 直接税和间接税的优缺点对比
      
      直接税（所得税、相续税等）：
      - 优点：采用累进课税制度，高所得者负担较重的税收，保证了所得差距的公平性。法人税采用固定课税。
      - 缺点：根据职业种类，难以正确掌握所得额，即使同等所得也会产生税额差异。随着工作收入增加而被征收更多税金，可能降低劳动意愿。
      
      间接税（消费税等）：
      - 优点：不论收入如何，负担相同；国家能获得稳定的税收。
      - 缺点：低收入者在收入中用于消费的比例较大，税负担较重，容易产生逆进性。
      ```

      ```markdown
      ## 公债
      
      公债是指国家和地方公共团体为了补充资金不足而发行的债务。
      
      ### 国债发行的原则
      
      国债分为建设国债（用于公共事业等建设）和特别国债（用于补充一般会计的资金不足）。然而，国债的发行受法律严格限制。国债的发行原则上是被禁止的，但建设国债作为特例被允许。不过，特别国债每个会计年度都要制定财政特例法才能发行。此外，日本银行认购国债也被原则上禁止，而是采用国债先卖给金融机构，再由金融机构卖给一般投资者的市场消化原则。
      
      ### 国债发行的问题点
      
      大量发行国债会带来以下问题：
      
      - 通货膨胀的可能性：为了偿还国债而大量发行货币，可能导致通货膨胀
      - 财政的弱化：由于国债偿还导致债务增加，社会保障相关支出等预算减少，影响财政服务的质量
      - 挤出效应：由于大量发行国债导致政府占用资金，导致民间可投资的资金减少
      - 世代间的不公平：国债需要由未来世代偿还，造成世代间的不公平

      建设国债从1965年开始发行，这是以1963-64年奥运会后的不景气为契机。特别国债则是在第一次石油危机后的1975-89年间，以及1994年以后每年都在发行。
      ```
