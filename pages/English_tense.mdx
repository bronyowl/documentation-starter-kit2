# 常见时态

1. 我将为您创建一个详细的表格，包含九种时态的谓语变化形式以及它们的常见语义用法：

   | 时态       | 谓语变化                 | 常见语义情况                                                 |
   | ---------- | ------------------------ | ------------------------------------------------------------ |
   | 一般现在时 | 动词原形(第三人称单数+s) | • 表示经常性、习惯性的动作<br/>• 表示客观事实或真理<br/>• 表示现在的状态 |
   | 一般过去时 | 动词过去式               | • 表示过去某个时间发生的动作<br/>• 表示过去完成的事情<br/>• 叙述过去发生的事件 |
   | 一般将来时 | will/shall + 动词原形    | • 表示将来某时会发生的动作<br/>• 表示计划或打算<br/>• 表示预测 |
   | 现在进行时 | am/is/are + 动词ing      | • 表示此刻正在进行的动作<br/>• 表示当前暂时的状态<br/>• 表示近期的安排 |
   | 过去进行时 | was/were + 动词ing       | • 表示过去某时正在进行的动作<br/>• 作为过去事件的背景<br/>• 表示过去持续的状态 |
   | 将来进行时 | will be + 动词ing        | • 表示将来某时正在进行的动作<br/>• 表示预先安排好的事情<br/>• 表示将来某时的持续状态 |
   | 现在完成时 | have/has + 过去分词      | • 表示过去发生且与现在有关的动作<br/>• 表示持续到现在的状态<br/>• 表示过去的经历 |
   | 过去完成时 | had + 过去分词           | • 表示过去某时之前发生的动作<br/>• 表示两个过去动作的先后关系<br/>• 表示过去的过去 |
   | 将来完成时 | will have + 过去分词     | • 表示将来某时之前完成的动作<br/>• 表示将来的回顾<br/>• 表示对将来完成状态的预测 |

   特别说明：

   1. 每种时态都有其特定的语境和使用场合，上述表格列出的是最常见的语义情况。

   2. 时态的选择往往需要考虑：
      - 动作发生的时间
      - 动作的持续性
      - 动作与其他事件的关系
      - 说话人的视角和意图

   3. 在实际使用中，时态的选择还需要考虑：
      - 上下文语境
      - 时间副词的搭配
      - 句子的语气和语用功能

   4. 有些时态可能在特定情况下可以互换使用，具体要根据上下文来决定最合适的表达方式。
